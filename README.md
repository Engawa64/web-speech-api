## 使い方
### 発音を聞く(音声合成)
1. テキストボックスにロシア語を入力し、**決定ボタン**を押す。<br>
<img width="320" alt="sample2" src="https://user-images.githubusercontent.com/65398246/156813575-5e9707d2-17ca-4942-a762-c9c5b280e160.png">
2. 画面に文字が表示される。
<img width="480" alt="sample3" src="https://user-images.githubusercontent.com/65398246/156814534-9d206e46-416c-4c55-b756-79cf13f255a8.png">
3. **読み上げボタン**を押すと、3回読み上げられるので発音を確認する。
<img width="480" alt="sample4" src="https://user-images.githubusercontent.com/65398246/156815111-1931f5b9-9c27-4a0f-9cdb-027b01bfd172.png">
<br>
### 発音を練習する(音声認識・音声合成)
1. 発音を確認した後、マイクの画像をクリック。<br>
<img width="400" alt="sample2-1" src="https://user-images.githubusercontent.com/65398246/156816794-2b70dd1f-a754-4ae4-9629-788d0bd36429.png">
2. **許可**を押す
<img width="480" alt="sample2-2" src="https://user-images.githubusercontent.com/65398246/156816938-0f0f63d0-1eb9-4d9a-ae5b-7d23e08958bb.png">
3. 画像が切り替わり(画像参照)、マイクがオンになる。
<img width="400" alt="sample2-3" src="https://user-images.githubusercontent.com/65398246/156817156-aeb137b0-596c-4ae1-a9a3-33eb485d055b.png">
4.マイクに向かって発音する<br>
合っていたら同じ言葉が返ってくる。間違っていたらロシア語で何て言いましたか？と聞き返される。<br>
**認識した結果は画面下部にピンクの文字で表示されます。**<br>
<img width="480" alt="sample2-4" src="https://user-images.githubusercontent.com/65398246/156817480-45eb4240-277f-4d8e-91f4-055b061a4991.png">
